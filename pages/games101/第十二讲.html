# Geometry 3

Mesh Subdivision 细节更丰富，增加三角形，就是细分

Mesh Simplification 简化三角形，用更小的三角形，同时也保持模型

二次误差度量

Mesh Regularization

细分的方法 网格是三角形，可以使用Loop Subdivision细分

网格不是三角形，使用Catmull-Clark Subdivision细分

简化的方法 摊缩边，使用二次误差度量

遗漏的话题 Shadow mapping 关键信息：如果一个点不在阴影里，那么这点会被光源和摄像头看得见 经典的Shadow Mapping只能处理点光源 非0即1的阴影是硬阴影，还有软阴影

什么是硬阴影，就是阴影非常锐利，要么能看到，要么看不到

软阴影，不是非0即1，会有过渡 本影，半影，软影就是它们的过渡
